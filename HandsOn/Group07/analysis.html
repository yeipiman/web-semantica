Resource Naming Strategy

For this project, we have defined the following base URI for our ontology and data:

<http://movemadrid.org/def/base#>

All classes and instances will follow a consistent pattern:

  • Classes: BaseURI#ClassName (e.g., http://movemadrid.org/def/base#CulturalEvent)
  • Instances: BaseURI#InstanceName (e.g., http://movemadrid.org/def/base#Evento50066046)

This ensures that each resource is uniquely identifiable and consistent across our datasets.
