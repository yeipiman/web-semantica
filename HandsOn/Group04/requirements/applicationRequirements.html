Application Requirements

Application Description

  • The application will show users all municipal facilities that offer free WiFi (libraries, cultural centers, sports centers, etc.).
  • The application will display the main features of each facility, such as name, address, district, neighborhood, and opening hours.
  • Users will be able to filter facilities by criteria such as district, neighborhood, type of facility, or hours of availability.
  • These data can be combined with other datasets (for example, population or tourism) to display additional metrics such as coverage or accessibility.
