Dataset Requirements

Selected Dataset

Municipal facilities with free WiFi zones

R1: Data in Smart City Domains

The dataset relates to public WiFi zones in municipal facilities (libraries, cultural centers, etc.), which are part of the smart city domain “connectivity” and “public services”.

R2: Data in CSV Format

The dataset is available in CSV format, making it easy to use and process.

R3: Open License

The dataset is published with an open license (Creative Commons Attribution or equivalent) that allows reuse and publication.

R4: Linkability with Real-World Entities

The dataset includes information about locations of WiFi zones (addresses, districts, neighborhoods, coordinates). These can be linked with other geographic datasets, demographic data, or municipal service maps.

R5: Documentation (Optional)

Documentation is available on the datos.gob.es portal, including descriptions of fields, dataset structure, and update frequency.

R6: Multiple Data Sources (Optional)

While this dataset originates from municipal records, it can be enriched by combining it with other sources, such as population density datasets, tourist maps, or municipal service directories, to create a more comprehensive smart city application.
