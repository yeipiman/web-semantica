Requirements Assessment for Selected Datasets

This document assesses whether two selected datasets meet the requirements for use in smart city projects. Requirements R1–R4 are mandatory; R5–R6 are optional and are discussed for each dataset.

Dataset 1: Censo de animales domésticos por distrito

  • Title: Censo de animales domésticos por distrito
  • Source: Comunidad Autónoma de Madrid, Colegio Oficial de Veterinarios de Madrid
  • Sector: Salud (Health)
  • Coverage: Districts of Madrid
  • Time range: 01/01/2014 – 31/12/2024
Requirement Description Satisfied? Notes
R1 Data are in a smart city domain. Yes Urban health and pet demographics.
R2 Data are available as a CSV file. Yes CSV format confirmed.
R3 Data have an open license for publication. Yes Open license confirmed.
R4 Data can be linked with real-world entities (e.g., locations). Yes District-level data enables linkage to locations.
R5 (optional) Documentation exists for the data. Yes Descriptive metadata and documentation available.
R6 (optional) Data comes from multiple sources. Yes Multiple institutions involved.

Dataset 2: Accidentes de tráfico de la ciudad de Madrid

  • Title: Accidentes de tráfico de la ciudad de Madrid
  • Source: Policía Municipal de Madrid
  • Sector: Urban mobility and safety
  • Coverage: City of Madrid
  • Time range: 2010–2018 (with structure change in 2019+)
Requirement Description Satisfied? Notes
R1 Data are in a smart city domain. Yes Traffic accidents: urban mobility and safety.
R2 Data are available as a CSV file. Yes CSV format confirmed via portal.
R3 Data have an open license for publication. Yes Open license confirmed via portal.
R4 Data can be linked with real-world entities (e.g., locations). Yes City-level data, some location linkage possible (not by neighborhood).
R5 (optional) Documentation exists for the data. Yes Associated documentation available.
R6 (optional) Data comes from multiple sources. No Mainly from Policía Municipal. Not aggregated from multiple sources.

Summary

Both datasets fully satisfy all mandatory requirements (R1–R4) for use in smart city applications. The first dataset ("Censo de animales domésticos por distrito") also satisfies both optional requirements (R5 and R6). The second dataset ("Accidentes de tráfico de la ciudad de Madrid") provides documentation (R5) but does not aggregate data from multiple sources (R6).

These datasets together offer valuable insights into urban health, demographics, and safety for the city of Madrid.
