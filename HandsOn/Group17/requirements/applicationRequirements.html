Application Requirements

1. Application Description

This app will allow users to search for different parks, gardens, green zones or tree species in madrid.

2. User Interface description

  • The app will provide a list with many green zones, parks or tree species in Madrid.
  • For the trees, there will be an option to show a more detailed description showing the name of the park where its located, scientific name, the amount of them and an image if possible.
  • For the parks, gardens and green zones there will also be an option to show a more detailed description showing the tree species in them and the total amount of each one.
  • When trying to search for something it can be filtered by parks, green zones, gardens and trees.
