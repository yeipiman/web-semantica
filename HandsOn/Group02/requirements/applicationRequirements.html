Application Requirements

Application Goal

The goal of the application is to provide an easy-to-use interface where users can quickly find upcoming tourist activities in Barcelona.

Users should be able to perform simple searches and view relevant activity details in a clear format.

Functional Requirements

  • Users should be presented with an initial wellcome screen with a button to start the search.
  • Users should be able to search for activities by text input (e.g., "concert", "exhibition", "sports").
  • Users can filter activities search by date: today, this week, or select a specific date.
  • Users can filter activities search by location within Barcelona.
  • Activities search must display a list of activities, each containing the fields "Title", "Date and Time" and "Location".
  • Users should be able to click on an activity result to open a detail page of the activitie.
  • Users on the detailed page of an activitie should be able to click on a "Back" button to comeback to search results.
  • The detailed page should display all available information about the activity.
