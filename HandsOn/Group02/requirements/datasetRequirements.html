Dataset Requirements

R1. Data are in one of the selected smart city domains

This dataset belongs to the "Economy & Tourism" domain, which is one of the smart city domains since tourism directly impacts urban planning, mobility, and local economy. It supports smart city goals by promoting sustainable tourism and linking cultural, social, and environmental aspects.

R2. Data are available as a CSV file

The open data portal of Barcelona Diputación provides the agenda in CSV format.

R3. Data have an open license so they can be published

The Diputación de Barcelona open data portal uses an open license (CC BY 4.0), which allows reuse, publication, and sharing as long as attribution is given. Therefore, the dataset can be republished for academic or public use.

R4. Data can be easily linked with generic real-world entities (e.g., locations)

The dataset contains information about different tourist activities, these activities have different information referring real-world institutions, contact information, location information, URLs... which are easy to model and link. For example, the field "Organitzadors" can be matched with a Wikidata URI of the given institution, making the dataset easily linkable.
