Barcelona Activities Dataset – Analysis

Dataset OpenRefine Manipulation

1. Translation of Column Names

The dataset originally contained column names in Catalan. We translated them into Spanish to improve readability and ensure the schema is consistent with the local context (Barcelona).

Example translations:

  • “Títol” → “Título”
  • “Data” → “Fecha”
  • “Ubicació” → “Ubicación”

2. Data Clustering and Cleaning

OpenRefine’s clustering functions were used to detect inconsistent or duplicated values in textual fields (e.g., activity names, location names, categories). Both key collision and nearest neighbor methods were applied to unify variations of the same entity.

Example: “Museo Picasso” and “Museu Picasso” were merged into one consistent value.

3. Column Cleaning and Refinement

Columns were analyzed and removed or modified to improve clarity:

  • Removed duplicated columns: “Temas”, “Etiquetas”, “Relación con el punto”, “Pie de imagen”, “Observaciones”
  • Removed empty columns: “Colaboradores”, “Correo de contacto”
  • Removed irrelevant columns: “Categorías”

Column adjustments:

  • Moved “Público” values into “Dirección” and deleted “Público”
  • Renamed: “Documentación” → “Localización evento”, “Localización” → “Localización organizadores”

Data Analysis

Seasonality

The graph shows a concentration of activities between July and October, peaking in October. Early months show lower activity.

Price Distribution

Most activities are priced between €0 and €20, with secondary peaks at €40 and €100. Average price: €35.68.

Organizer Frequency

The most active organizers are “Consorci del Patrimoni de Sitges” (41 activities) and “Turismo Vivencial” (27 activities). Paid activities represent 60.26% of the dataset.
