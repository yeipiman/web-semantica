Requisitos del dataset “Calidad del aire — Ayuntamiento de Madrid”

Este documento detalla cómo el conjunto de datos elegido satisface cada uno de los requisitos R1 a R6, usando como fuente la página oficial del portal de datos abiertos del Ayuntamiento de Madrid. (datos.madrid.es)

R1. Dominio smart city → ambiente / calidad del aire

Se cumple. El dataset está claramente en el dominio de medio ambiente / calidad del aire, que es un dominio típico de ciudad inteligente (smart city). En la ficha se indica que su “sector” es “Medio ambiente”.

R2. Disponible como archivo CSV

Se cumple. En la página de descargas se ofrece explícitamente una versión CSV del conjunto de datos (“Calidad del aire. Tiempo real csv”) entre los formatos disponibles.

R3. Licencia abierta

Se cumple. La ficha indica que el conjunto de datos está bajo una licencia pública (vinculada al aviso legal del portal de datos abierto). Esto permite su reutilización y publicación, bajo los términos del portal de datos abiertos del Ayuntamiento de Madrid.

R4. Enlaces con entidades del mundo real (geoespacialmente, estaciones)

Se cumple. Los registros del dataset provienen de estaciones de medición georreferenciadas. Las estaciones tienen ubicaciones físicas identificables y los datos pueden asociarse con coordenadas o identificadores de estación para enlazar con mapas, barrios, distritos u otras fuentes espaciales.

R5. Documentación disponible (opcional)

Se cumple. En la página del portal aparece un fichero PDF titulado “Interpretación datos de calidad del aire” como documentación asociada al dataset.

R6. Posibilidad de integrar múltiples fuentes (opcional)

Se cumple. Aunque este dataset es publicado por una única fuente (Ayuntamiento de Madrid), es integrable con otros conjuntos de datos públicos (por ejemplo, datos meteorológicos, mapas municipales, datos de salud, tráfico, etc.). Es decir, puede combinarse con múltiples fuentes externas como parte de la aplicación.

Referencias: “Calidad del aire. Datos en tiempo real” — portal datos abiertos Ayuntamiento de Madrid.
