Madrid Air Quality — Application Requirements

Description

This web application allows users to explore hourly air quality data in Madrid. Users can browse monitoring stations, view pollutant levels, and export data in RDF or CSV.

Main Requirements

  • View list of monitoring stations and their main pollutants.
  • Show hourly measurements for a selected station and date.
  • Provide export options in CSV and RDF (Turtle).
