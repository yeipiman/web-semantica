Application - Museums of Madrid Explorer

Description

Museums of Madrid Explorer is a web application guiding users through Madrid’s museums like the Prado and Reina Sofía. It offers an interactive map and general information for each museum.

Objectives

  • Display an interactive map of Madrid’s museums.
  • Visualize highlights and exhibit details.
  • Provide real-time updates on exhibitions, tickets, and hours.

Requirements

Functional Requirements

  • FR1: Show an interactive map with museum locations.
  • FR2: Filter museums by type or proximity.
  • FR3: Allow offline access to guides and itineraries.

Non-functional Requirements

  • NFR1: Responsive and accessible interface.
  • NFR2: Multilingual support (English, Spanish, French).
  • NFR3: Automatic weekly data updates.
