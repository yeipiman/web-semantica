Practical notes for reuse

  • Attribution: Because of CC‑BY, reuse must include attribution to EMT / EMT data portal.
  • Spatial joins: Use lat,long to join to GIS layers (GeoJSON/KML resources are also available in the same dataset for convenience).
  • Time dimension: This CSV is a static snapshot listing aparcamiento attributes. If you need historical availability or occupancy you should use the time-series parking availability datasets (published separately by EMT).
  • Combining sources (R6): To create multi-source datasets, combine these records with municipal registries, traffic sensors, or parking transaction logs — ensure license compatibility and preserve attribution.
